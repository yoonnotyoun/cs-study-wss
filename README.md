## How It Works

1. 매주 월요일 밤 9시 디스코드에서 스터디 진행
2. 스터디 전, 합의한 범위 공부하고 각자 맡은 파트를 notion에 정리
3. 스터디 전, 해당 범위에 해당하는 연습문제 풀어 github에 commit
4. 스터디 중, 각자 정리해온 파트와 연습문제 리뷰하고 서기가 한 파일로 정리해서 commit
