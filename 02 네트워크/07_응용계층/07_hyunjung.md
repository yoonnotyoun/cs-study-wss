3. 

   ```
   [HTTP 1.0]
   - 전송받을 문서에 이미지가 있으면 문서를 받을 때와 이미지를 받을 때 각각 연결을 설정한다.
   - 문서에 이미지가 포함되어 있으면 연결 과정을 두번 거쳐 전체 8단계로 모든 작업을 완료한다.
   [HTTP 1.1]
   - 전송받을 문서에 이미지가 있으면 다시 연결을 설정하지 않고 연결된 소켓을 통해 데이터(이미지)를 전송받는다.
   - 6단계로 모든 작업을 완료한다.
   -> 프로토콜 수행 성능이 향상되었다.
   * HTTP 2도 있다.
   ```
