# 02 네트워크의 구성

*<네트워크 개론> pg 103~105*



## 연습문제

1. 

   ```
   근거리 네트워크(LAN)
   - 범위가 건물 안이나 특정 지역인 네트워크로 유선 케이블, 적외선 링크, 무선 송수신기 등을 이용하여 통신
   
   캠퍼스 네트워크(CAN, 인트라넷의 일종)
   - 인터넷에서 사용하는 회선과 여러 기반 기술을 이용하여 구축하는 사설 네트워크
   - 각 캠퍼스에서 가장 가까운 ISP까지 연결하면 인터넷을 이용하여 저렴한 비용으로 사설 네트워크 구축 가능
   
   광역 네트워크(WAN)
   - 2개 이상의 LAN을 넓은 지역에 걸쳐 연결한 것
   - ISP가 제공하는 서비스를 사용하여 구축된 네트워크
   ```

2. 

   ```
   LAN카드(NIC)
   ```

3. 

   ```
   더미 허브
   - 단순히 컴퓨터와 컴퓨터 간의 네트워크를 중계하는 역할을 함
   - 네트워크의 전체 대역폭을 노드 수만큼 분할해서 사용하기 때문에 연결된 노드 수가 증가하면 네트워크의 속도가 떨어짐
   
   스위칭 허브
   - 단순 전송 기능을 넘어 수신지 주소로 스위칭하는 기능 있음
   - 노드들을 각각 점대점으로 접속시키기 때문에 네트워크의 효율 높음
   ```

4. 

   ```
   브리지
   - 2개 이상의 근거리 통신망을 연결하여 하나의 네트워크로 만드는 장치
   - 수신지 주소에 따라 특정 네트워크 트래픽만 통과시킬 수 있도록 설계된 특수한 형태의 네트워크 스위치
   - 받은 신호를 증폭하고 입력과 출력을 연결 (스위치와 비슷)
   - 전체 네트워크의 트래픽을 줄이기 위해 네트워크를 세그먼트 단위로 분할
   
   라우터
   - 구조가 다른 망을 연결할 수 있는 장치
   - IP주소를 바탕으로 데이터가 수신지까지 갈 수 있는 경로를 검사하여 효율적인 경로를 선택하는 라우팅 기능 수행
   - 인터넷상에서 수신지의 주소를 토대로 경로상의 다음 주소를 결정하여 그 경로로 데이터를 보냄
   ```

5. 

   ```
   AP는 유선 이더넷 네트워크에 무선 액세스를 제공
   (허브나 스위치,) 유선 라우터에 연결되어 무선 신호를 전송
   위치를 이동하면서도 계속해서 네트워크에 무선으로 액세스할 수 있음
   ```

6. 

   ```
   1) 성형
   - 장점: 설치 비용이 저렴, 유지 보수나 확장이 용이, 링크 하나가 끊어졌을 때 다른 링크는 영향을 받지 않음
   - 단점: 중앙에 있는 전송 제어 장치에 문제가 발생하면 전체 네트워크가 작동할 수 없음, 통신량이 많으면 전송 지연, 많은 케이블 연결
   
   2) 버스형
   - 장점: 간단한 설치, 케이블 비용 저렴, 장치를 추가하기 쉬움, 고장나도 전체 네트워크에 영향 미치지 않음, 중추 케이블을 효과적으로 설치 가능
   - 단점: 장비 수가 많아지면 네트워크 성능 저하, 중앙 케이블 고장나면 전체 네트워크가 작동하지 않음, 버스 케이블에 결함이나 파손 발생 시 모든 전송 중단, 재구성이나 결합 분리 어려움, 데이터 전송 시 충돌 발생 가능
   
   3) 트리형
   - 장점: 제어가 간단, 네트워크 확장 쉬움, 장비 간 데이터 전송 거리 늘릴 수 있음, 여러 컴퓨터를 분리하거나 우선순위 부여 가능
   - 단점: 중앙에 트래픽 몰려 병목 현상 발생 가능, 중앙의 전송 제어 장치가 다운되면 전체 네트워크 장애 발생
   
   4) 링형
   - 장점: 구조 단순, 설치 재구성 쉬움, 장애 발생 시에도 복구 빠름, 한 장치가 특정한 시간 내에 신호를 받지 못하면 경보, 케이블 비용 저렴
   - 단점: 링을 제어하는 절차 복잡, 새 장비 연결하려면 링을 절단 후 추가해야 함, 단순 링형의 경우 링에 결함이 발생하면 전체 네트워크 사용 불가
   
   5) 그물형
   - 장점: 전용 링크를 통해 데이터 전송하므로 통신량 문제 해결, 한 링크가 고장나더라도 전체 시스템에는 영향 없음, 모든 메시지를 전용선으로 보내기 때문에 비밀 유지나 보안에 유리, 결함의 식별과 분리가 쉬움
   - 단점: 설치와 재구성 어려움, 필요한 전선의 용적이 큼, 네트워크가 복잡하고 많은 통신 회선이 필요해 비용 높음
   ```

7. 

   ```
   - 데이터를 전송할 수 있는 최대 전송 속도
   - 네트워크에서 이용 가능한 신호의 최고 주파수와 최저 주파수의 차이
   - 기본 단위는 bps
   ```

8. 

   ```
   공통
   - 데이터 전송 속도 10Mbps
   - 전기적 간섭에 영향 받지 않음
   
   10BASE-2
   - 최대 185m
   - 쉽게 구부러져 사무실의 컴퓨터에 직접 연결 가능
   - 설치 쉬움
   - 설치 비용 저렴
   
   10BASE-5
   - 최대 500m
   - 쉽게 구부러지지 않아 드롭 케이블을 이용하여 연결
   - 설치 어려움
   - 설치 비용 비쌈(광섬유 케이블보다는 저렴)
   ```

9. 

   ```
   베이스밴드 동축 케이블
   - 디지털 신호 전송(LAN) 용도
   - 수백 m~1km 까지 전송
   - 수십 Kbps~500Mbps 속도로 전송
   
   브로드밴드 동축 케이블
   - 아날로그 신호 전송(케이블 TV)용도
   - 수십km 까지 전송
   - 수십 Kbps~수백 Mbps 속도로 전송
   ```

10.  

    ```
    1) UTP 케이블
    - 전기적 신호의 간섭을 줄이기 위해 케이블을 나선형으로 꼬아놓은 형태
    - 별도의 피복 없음
    - 가격이 저렴하고 쉽게 만들 수 있기 때문에 네트워크 케이블의 90% 이상을 점유
    - 근거리 통신망을 구성하는 전송 매체
    
    2) FTP 케이블
    - 전기적 신호의 간섭을 줄이기 위해 전체 케이블에 피복을 씌운 형태
    
    3) STP 케이블
    - 전체 케이블에 피복을 씌우고 알루미늄박 등의 금속 물질로 한 번 더 감싼 형태
    - 차폐재가 접지 역할까지 해 외부 잡음을 차단할 수 있으며 전기적 신호 간섭에 좋은 성능을 보임
    ```

    ```
    공간으로 전파되는 전자파를 매개체로 데이터를 전송
    - 라디오파: 빛의 속도로 데이터를 전송하면서도 진공 상태나 대기 통과 가능
    - 마이크로파: 주파수가 매우 높은 전파
    ```

